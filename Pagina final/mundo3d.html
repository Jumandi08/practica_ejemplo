Mundo 3D

Es el proceso de desarrollo de una representación matemática de cualquier objeto tridimensional a través de un software especializado. Al producto se le llama modelo 3D,Con este tipo de arte digital los modelos son representados en el computador como objetos tridimensionales mediante el uso de puntos que contienen información dentro de un espacio 3D, conectados por varios elementos tales como vertices, filos, caras, líneas, superficies, curvas, etc.
